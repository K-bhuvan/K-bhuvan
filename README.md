## 🔥 Latest Projects

### 🧠 [LLM From Scratch](https://github.com/K-bhuvan/llm-from-scratch)
> Training a Large Language Model from the ground up to deeply understand transformer architectures, tokenization, and training dynamics.

- Built and trained transformer-based language models end-to-end
- Explored architecture design choices and optimization strategies
- **Tech:** Python, PyTorch

---

### 🚗 [Autonomous Driving with Reinforcement Learning](https://github.com/K-bhuvan/autonomous-driving-rl)
> RL-based autonomous vehicle control using state-of-the-art policy gradient algorithms.

- Implemented and benchmarked **PPO**, **SAC**, and **TD3** algorithms for vehicle control
- Trained agents in simulation environments for real-world-applicable driving behaviors
- **Tech:** Python, Reinforcement Learning, Gym/Simulation

---

### 👁️ [Stereo Visual Odometry (SVO)](https://github.com/K-bhuvan/Stereo-Visual-Odometry-SVO)

- Implemented full stereo visual odometry pipeline from scratch
- Applied feature detection, matching, and pose estimation techniques
- **Tech:** Python, OpenCV, NumPy

---

### 🏭 [Warehouse Automation](https://github.com/K-bhuvan/project1_warehouse_automation)
